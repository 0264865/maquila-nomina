# Sistema de Nómina para Maquiladora Textil

Aplicación web para automatizar el cálculo de nómina a partir de archivos Excel del mostrador.

## Características
- Registro de trabajadores con cálculo automático de sueldo diario y por hora según ley mexicana
- Procesamiento de archivos Excel del mostrador
- Cálculo de horas trabajadas exactas (horas y minutos)
- Generación automática de reportes de nómina
- Exportación a Excel
